Проект быстрого обучения слепой печати

Данный проект находится в режиме пробной эксплуатации. Регистрация происходит без подтверждения электронного адреса, без гарантии хранения регистрации длительное время. Данные могут быть сброшены, в ходе развития проекта.

Цель проекта реализовать платформу для обучения слепой печати на основе методики Б.Ц. Бадмаев, которая в свою очередь основана на теории П.Я. Гальперина.

Основа этой методики заключается в ускорении формирования устойчивого навыка при активации нескольких зон коры головного мозга, данном случае зон ответственных за мелкую моторику и зоны ответственную за речь (центр Брока).

Данная методика позволяет сократить время обучения слепой печати десятипальцевым методом на порядок, сформировать навык возможно за 3-5 часов.

Особенностью данного подхода, является легкость применения методики обучения на другом языке, для освоения английской раскладки, после пройденного курса на русской, требуется полчаса-час.

Данная особенность открывает перспективы легкого освоения альтернативных раскладок на разных языках. Временные затраты на обучение новых, альтернативных раскладок не идет ни в какое сравнение с классическим методами формирования моторных навыков слепой печат.

Данная методика может быть применена независима от текущего проекта с использованием сторонних клавиатурных тренажеров, и в том числе вовсе без специальных тренажеров, в обычном в текстовом редакторе.

Для генерации текста используется Марковские сети. Генератор работает в двух режимах, режим генерации по символьный и по слову. Также можно менять количество предыдущих символов или слов влияющих на выбор следующего символа или слова. Данное количество варьируется от 1 до 5. Для 1 учитывается только пары элементов расположенные - ядом сгенерированный текст для символов выглядит как абракадабра, для слов как случайный набор слов. Для параметра равного 5, для генератора на основе символов формируются слова, для генератора на основе слов, предложения, которые иногда могут быть практическими цитатами из произведений Л.Н. Толстого.

В ближайшем будущем, проект будет оставаться бесплатным для частного использования, при условии, что нагрузка не потребует расширения облачных ресурсов.

Используется подсказка в виде команды для пальцев, в виде текста и графических знаков. Данная особенность не входит в методику Б.Ц. Бадмаева, и влияние ее на скорость обучения не исследовано. На первом этапе она помогает пользователям, для понимания методики. После того как пользователь освоил саму методику, подсказку имеем смысл отключить.
